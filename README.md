# Home Page

## Descripción

Este proyecto consiste en una página web de inicio que permite a los usuarios registrarse o iniciar sesión para acceder a una página principal donde podrán visualizar y organizar accesos rápidos a sus aplicaciones y sitios web favoritos.

La aplicación incluye un formulario de registro con validación básica de datos y opciones de inicio de sesión mediante diferentes proveedores como Google, Facebook y Microsoft. Su objetivo es centralizar en una sola página los enlaces más utilizados por el usuario para facilitar un acceso rápido y cómodo a sus herramientas favoritas.

## Tecnologías utilizadas

- HTML5
- CSS3
- Bootstrap Icons

## Aprendizajes aplicados

Durante este proyecto se practicaron los siguientes conceptos:

- Estructura básica de HTML5.
- Formularios.
- Etiquetas `label`.
- Inputs de tipo:
  - `text`
  - `email`
  - `password`

- Enlaces (`a`).
- Clases CSS.
- Inclusión de librerías externas mediante CDN.
- Uso de iconos tipográficos con Bootstrap Icons.
- Organización de archivos CSS.
- Asociación entre etiquetas `label` e `input` mediante `for` e `id`.
- Uso de atributos de validación como `required`.
